# fumufumu

モノレポ構成のNext.js + Cloudflare Workersプロジェクト

## 概要

- **フロントエンド**: Next.js 15 (App Router)
- **バックエンド**: Cloudflare Workers + Hono
- **パッケージマネージャー**: pnpm
- **リンター**: Biome

## クイックスタート

```bash
# 依存関係のインストール
pnpm install

# 開発サーバー起動
pnpm dev
```

詳細は[docs/README.md](./docs/README.md)を参照してください。
