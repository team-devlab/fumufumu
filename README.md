# fumufumu

Next.js + Cloudflare Workersプロジェクト

## 概要

- **フロントエンド**: Next.js 15 (App Router)
- **バックエンド**: Cloudflare Workers + Hono
- **パッケージマネージャー**: pnpm
- **リンター**: Biome

## クイックスタート

各アプリは独立して動作します。

### フロントエンド

```bash
cd apps/fumufumu-frontend
pnpm install
pnpm dev
```

### バックエンド

```bash
cd apps/fumufumu-backend
pnpm install
pnpm dev
```
