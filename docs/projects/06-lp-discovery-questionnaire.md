# 現状fumufumuの雰囲気・カラー観察メモ（暫定）

## 1. 画面トーン
- 全体は「やわらかい・安心感・実務的」なUIトーン
- 白背景 + グレー階調の面構成が中心
- 情報を読みやすくするカードUI（角丸・薄い境界線・控えめシャドウ）を多用

## 2. カラー傾向
- ベース: `gray` 系（背景/境界/本文）
- 主アクセント: `teal` 系（プロフィール、状態バッジ、主要アクションの一部）
- 補助アクセント: `blue` 系（認証・新規作成ボタン等）
- 注意/補助: `amber` 系（下書き・注意喚起）
- エラー: `red` 系

## 3. 現状から読み取れる印象
- 攻撃的・派手というより、落ち着いたプロダクト感
- コミュニティ/相談系に合う「心理的安全性」を意識した配色
- UIスタイルは機能優先で、ブランド演出はまだ控えめ

## 4. LP設計に活かせるポイント
- 既存プロダクトとの連続性を保つなら、`teal` を主軸に据えるのが自然
- CTAは1色に寄せて明確化し、補助CTAは中立色で差をつける
- 既存が静かなトーンなので、LPではヒーロー領域だけ少し情緒的なビジュアル演出を足すと差別化しやすい

## Gemini回答
* **ユーザーの心理:** 「暇つぶしにタイムラインを眺めていたら、なんか広告が流れてきたからタップしてみた」
* **温度感:** かなり低め（ふらっと立ち寄っただけ）。
* **特徴と対策:** とにかく**「離脱率」が跳ね上がりやすい**媒体です。少しでも「文字が多い」「自分に関係ない」と思われると、1秒で画面を閉じられてしまいます。そのため、パッと見の直感的なわかりやすさや、先ほどの「柄入り背景」のようなポップで気軽なデザインが刺さりやすい傾向があります。
