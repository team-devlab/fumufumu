# 07. LP印象分析メモ（`/lp/worldview-v2`）

## 対象LP
- 画面URL: `/lp/worldview-v2`
- 実装ファイル: `apps/fumufumu-frontend/src/app/lp/worldview-v2/page.tsx`
- 目的: 「うつ病患者向けのHPに見える」印象が生まれる原因を、デザイン/コピー/導線の観点で整理する

---

## 結論（要約）
このLPは「安心・受容・静かな伴走」の表現が強く、キャリア意思決定サービスとしての業務文脈より先に、心理ケア系の文脈が前面に出る構成になっている。  
結果として、閲覧者によっては医療・福祉系サイトに近い第一印象を受けやすい。

---

## 原因整理（4観点）

### 1. コピーが心理ケア寄りに偏っている
- ヒーロー見出しが `「悩みを話す前から、安心できる」` を中核にしている
  - 参照: `page.tsx` 86-91行
- 本文でも `やさしく / 安心 / 気持ちが整う / 焦らせない` といった語彙の比率が高い
  - 参照: 94-96行、115-119行、49-55行
- 「キャリア判断」の具体（例: どの意思決定をどう支援するか）が、第一スクリーンで弱い

### 2. ビジュアル言語が医療・福祉系の定番に近い
- 淡いティール + 白背景 +大きな余白 + 角丸カード + 柔らかい影
  - 参照: 79-80行、126行、150行、182行
- セーフティカラー（teal/green）主体で、緊張感を下げるトーンが強い
- アイコン表現も抽象的で、職務文脈（エンジニア業務）を想起しにくい

### 3. 「誰のどんな悩みをどう解決するか」の業務文脈が薄い
- `Scene` に年収/転職/技術選択はあるが、全体の主語は「空気感」「安心」に寄っている
  - 参照: 109-119行、39-43行
- 世界観セクションも「受け止める」「安心できる場」が主で、意思決定支援の具体手段が見えにくい
  - 参照: 24-36行、141-143行
- そのため「キャリア判断サービス」より「心理的サポートサイト」に寄って見える

### 4. CTAが抽象的で行動イメージが湧きにくい
- CTA文言が `相談してみる` のみで、何が起きるか（投稿/閲覧/比較）が不明瞭
  - 参照: 103行、248行
- ユーザーの行動解像度が低いと、情緒訴求だけが印象に残りやすい

---

## 補足
- このLPの良さ（安心感・受容姿勢）は明確に出ているため、問題は「優しさの過不足」ではなく「業務文脈の比率」。
- 方向性としては、安心訴求を残したまま、具体課題の記述密度を上げるのが最も効果的。
